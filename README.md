# Volcano REST-API Service 

#### Run Integration Tests
    mvn test
    
#### Run the Project
    mvn spring-boot:run
    curl localhost:8080

#### Endpoints
     1. http://localhost:8080/
        1.1 [GET] => desc
     
     2. http://localhost:8080/reservations/availabledates
        2.1 [GET] => desc
     
     3. http://localhost:8080/reservations
        3.1 [GET] =>  desc
        3.2 [POST] => desc
        3.3 [PUT] => desc
        3.4 [DELETE] => desc
